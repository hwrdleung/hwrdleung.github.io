About Me:

I started learning web development over at freecodecamp.org and various other online resources about one year ago. What I love about web development is that it is a skill which merges art and science--every solution requires some logical thinking as well as creativity. There are many factors to consider with each project, all of which have a direct impact on the quality of the finished product. These are some of the things that I always keep in mind when I code:

User experience

I strive to create a positive user experience with each of my projects by designing layouts that are easy on the eyes, effortless to navigate, and guides the user where necessary.

Maintainability

Writing simple, organized, and well documented code helps to ensure that updates and changes are made with greater ease.

Performance

I aim for optimal performance by writing code that functions efficiently and by compressing media files like images and videos as much as possible to reduce load times for users.
